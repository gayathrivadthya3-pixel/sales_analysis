# Sales Data Analysis Project

## Overview

This project demonstrates how to perform basic data analysis using Python and pandas on a real-world sales dataset. It includes data loading, cleaning, exploration, and generating meaningful business insights.

---

## Objectives

* Understand the basics of data analysis
* Work with CSV data using pandas
* Perform data cleaning and preprocessing
* Calculate key business metrics
* Generate a simple analysis report

---

## Technologies Used

* Python
* Pandas
* NumPy

---

## Project Structure

```
sales_analysis.py      # Main Python script
sales_data.csv         # Dataset file
analysis_report.md     # Final analysis report
requirements.txt       # Required libraries
```

---

## Features

* Load and explore dataset
* Handle missing values and duplicates
* Perform statistical analysis
* Identify best-selling products
* Generate formatted output

---

## Key Metrics Calculated

* Total Revenue
* Average Sales
* Highest Sale
* Best Selling Product

---

## How to Run

### Step 1: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 2: Run the project

```bash
python sales_analysis.py
```

## Data Cleaning Steps

* Filled missing numeric values using mean
* Replaced missing categorical values with "Unknown"
* Removed duplicate records
